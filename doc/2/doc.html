劇情與角色設定

一、概述

此處為說明遊戲整體劇情，其中包含了遊戲流程所接觸到的劇情、世界觀與角色設定等。

二、劇情發展

（一）、前導劇情

序幕
  1. 故事發生在現代世界，主角是一名普通的上班族（程續緣）。
  2. 某天當主角下班後走在路上時，他看到了一隻「渡鴉」。不知為何湧現一股熟悉感的他，覺得必須要追上去一探究竟，因此立刻追了上去。但是當他轉入一個轉角時，一道白光突然壟罩了他的雙眼...
  3. 白光就是煞車不及的車前燈燈光。日復一日的主角對「活著」早已麻痺、更不知何為「死亡」。意識朦朧的狀況下被車擦撞後入院。身體機能上並無大礙，但進入了昏迷狀態。
  4. 入院後主角的意識沒有甦醒的跡象，因此醫師判斷需要使用特殊的療法－「甦醒醫療技術」（關於該技術的設定可見「三、主世界觀與設定」）。
  5. 醫師告訴主角母親：主角必須從精神世界（內心）中找到關鍵性的記憶（夢想），也就是真正渴望、嚮往、想做的事，才能夠讓主角甦醒。母親對此的反應是：飛行是小時候的夢想，他認為主角已經找到另外的夢想，因此沒有特別想阻止這場治療（但主角內心深處的夢想仍然是飛行）。
主角人生經歷
  1. 出生家庭：主角生於一個溫馨的家庭，身為獨生子的他有一父一母。父親是飛機工程師，也是一名業餘的飛行員。他擁有一架小型的私人民用飛機，四處飛行探索世界是他的興趣之一。母親則是家庭主婦。
  2. 童年探索：父親在時常和主角分享飛行的快樂，也會教導主角許多關於飛行的知識。主角平常有記筆記的習慣，記下這些知識的同時，更是會閱讀科學與歷史（有名的飛行員或是飛行技術的發展）。記筆記習慣的源由是父親教導要做「飛行日誌」，就像一名飛行員一樣。童年的主角對「飛行」產生相當的憧憬－夢想著當個飛行員（私人飛機駕駛員），有一架屬於自己的飛機，並且可以自由自在地探索整個世界。
  3. 歷經創傷：小時候某次父親帶母親與主角一同飛行，但是途中遭遇了飛行事故。原因是當時天氣狀況不佳，某隻渡鴉碰撞到飛機導致機械失靈。飛機迫降的過程中父親想盡辦法保護母子，雖然飛機成功迫降於某座海島的臨海上，但父親卻因而往生。渡鴉碰撞發出的叫聲，也成為驚恐下主角的創傷記憶點。最後母子流離到海島上，成功獲救。
  4. 家庭劇變：事故後，出於對「飛行」的恐懼，母親嚴厲禁止主角接觸有關飛行的事物。同時主角也處於創傷後的壓力之下，心裡塵封了飛行的夢想。在沒有夢想的加持下，主角無以發揮繼承自父親的才華。母親向主角灌輸「追求穩定生活」的思想，父親的往生少了給予夢想的因素。主角逐漸被潛移默化，也是這兩項原因導致了主角逐漸遺忘夢想。
  5. 逐漸成年：學生時代以考試導向的教育體制讓主角在成長過程中漸漸迷失方向，只會學習別人限制的知識。而弱肉強食的社會體制更是讓主角在工作時更難接觸到自己有興趣的事物，常常因為工作機會考量而強迫自己去做不想做的事。
  6. 現況：在索然無味的日常生活中，成為一名普通上班族。每天加班卻沒有目標，也找不到人生的意義。最終由於身體狀況變差以及心理狀態的日漸貧乏，最後趨近於麻痺的狀態，只是活著的、會動的低產能機器。當長大後的主角回想到自己曾經的夢想時為感到很後悔，但是自己已經錯過人生的精華階段，不可能再重來。感到一絲帶有遺憾的暖意，俗話說「花有重開日，人無再少年。」大概就是指這種情況吧。

（二）、遊戲過程

遊戲隨著三關代表著主角不同的探索歷程：脫離現實（一）、打敗心魔（二）、找回夢想（三）。

第一關
  • 場景在一個島上，關卡場景會讓主角逐漸想起被他遺忘的夢想，最後乘著熱氣球離開島嶼。
  • 正面為貧乏無趣的現實。
  • 反面為童年放蕩不羈的想像，看似異常而且使人（成人／玩家）警戒，卻有著無限可能。
  • 意象為脫離現實、返回童年並想起關於飛行的回憶。遊戲方式並不是屏棄「成年現實」，而是必須靠兩面的合作才能通過關卡。也象徵著最終結局是「成年現實」與「童年夢想」的契合與平衡。
第二關
  • 場景在天空之城（浮於空中的城堡），遇到不同的NPC後讓主角更加明白自己與過去的區別。
  • 正面天空之城的意象代表童年的異想世界，反面的重複無意義勞動與生產工廠就是成年。
  • 關卡過程中，藉由與童年象徵的NPC互動並重拾童年記憶，在兩面的協調中完成任務。
  • 意象為面對現實、擊潰黑暗並完成「自己」的寄託。關底需要面對並打破成年的黑暗（心魔）。
第三關
  • 場景在海上，即為當年飛機失事墜毀的海洋。主角將在此回憶起所有往事。
  • 正面雷雨交加，反面則是一片晴朗。象徵成年的現實與幼年的理想，在飛行的過程都會遇到，並無孰優孰劣的問題，而是並存於道路上的境遇。
  • 意象為重拾夢想並且試著完成，回到當年的事故現場（創傷來源）且重新開始。

（三）、結局劇情

最終結局將根據購買道具的不同，而進入到不同的結局（使用對話進行引導）。

  • Bad End（購買飛行手冊）：飛機最後再次墜機，原因是主角不知道該往哪邊前進（尚未找回關鍵記憶／夢想），因此不慎捲入了亂流。最後醫生會詢問是否要再來一次。
  • Good End（購買護目鏡）：飛機飛離世界（精神世界）表示主角找回關鍵記憶／夢想，主角成功甦醒並解除了心中的陰霾。結局還是回去當普通的上班族。
  • True End（購買地圖）：飛機飛離世界（精神世界）、主角成功甦醒。此後向母親傾訴自己真正的夢想其實是想當上飛行員，最後開飛機載著母親抵達了當初父親沒能帶他們去到的地方－天空之城（白露里智奧，關於天空之城設定可見「三、主世界觀與設定」）。

三、主世界觀與設定

甦醒醫療技術
  1. 屬於科幻的醫療技術，現實中並不存在。
  2. 甦醒醫療技術大多用在幫助昏迷的人們找回自己的意識，可以給受到外力創傷或精神衝擊的人治療，並幫助他們「甦醒」至正常狀態。
  3. 但條件是該患者在治療期間，必須回想起關鍵性的記憶（遊戲中設定為夢想），若成功回想起則可順利從現實中甦醒。
  4. 若失敗不會死亡、仍會持續昏迷，可以有重來的機會。但失敗一次意味著該患者透過這種方式甦醒的機會很低，因此一般不會多次嘗試。設定上使用該技術也需要金錢成本。
  5. 這個技術只有對身體機能甦醒後可運作，但由於失去意識而昏迷的人有效。
遊戲世界

透過甦醒醫療技術，主角可以進入到自己的精神世界，也就是玩家操控主角所在的「遊戲世界」。正面是藉由主角成年後人格所「感受、理解到的世界」；反面則是主角童年時人格所「感受、理解到的世界」。因此，正面世界的場景與物件自然會比較貼近現實（因為是成年思維所看到的），而反面則較不受拘束（童年有很多的想像）。無論是哪一個世界，客觀來說都不屬於現實。在兩個世界切換時有以下幾點設定：

  1. 某些物件在正反面輪廓相同，而內容不同；甚至正反兩面可能一面為生物、一面為無生命。
  2. 玩家可以共有正反兩面的記憶（都是在精神世界內）。
  3. 正反兩面的「同一個NPC」並不知道彼此的存在，知識也不共有。
  4. 正反兩面切換時所有物件的「宏觀物理量」都會相同（例如：正面的石頭以5m/s的速率飛行，切換後的紙團應該也要以5m/s的速率飛行，即便考慮質量會造成兩者動能不同）。
天空之城－白露里智奧

天空之城原型為現實世界坐落於義大利的「白露里智奧」原型。遊戲世界設定為一群嚮往著天空的人們所建造的城堡。雖然沒能完成翱翔天際的夢想，卻也找到了高聳入雲的、離天空最近的一個夢想之地，興建他們的城市。父親當年開著飛機並載母子三人就是為了前進該地，但此次飛行因為失事而未抵達目的地，也成了父親最後一次的飛行（最後波紋）。

如果主角最後擁有雅努斯之石，會在劇情畫面中看到關於這顆石頭的消息：其實是他父親遺留下來的石頭。白露里智奧的岩石很珍貴和稀有，所以會引起人的好奇心。岩石的碎片被作為護身符，是因為其特別堅硬。象徵著在考驗之後淬煉出來的部分。

貨幣的意義

貨幣是天空之城居民所使用的貨幣，上有雙面神「Janus」的圖案，結合硬幣本身也可以翻轉再次強調「Flip」的意象。主角會在不同地點獲得貨幣，最後需要使用貨幣購買關鍵道具，將決定最終結局的走向。

墓碑符號

第一關的墓碑就是父親的，當年飛機失事後，母子漂流到小島上便為父親做了簡陋的墓碑（但當時並沒有墓碑符號）。遊戲世界（精神世界）中的墓碑出現了符號，原型為現實世界「白露里智奧」的飛行路線。象徵父親當時未能完成的飛行，因此才會出現在父親的墓碑上。

渡鴉的意義

渡鴉在現實世界被認為與死亡及亡魂有所關連。在瑞典，渡鴉被認為是被殺之人的鬼魂；在德國，牠們則是被咒詛之人的靈魂（維基百科 ）。遊戲中象徵了主角父親亡魂的引導。深藏於潛意識內的父親以渡鴉出現在世界中，幫助主角重拾童年的回憶並順利找回夢想（甦醒）。

渡鴉也可以象徵主角的創傷記憶點。遊戲中渡鴉是存檔點，也時常跟著玩家待在某些不顯眼的位置。但是一直都在，就如同主角童年的創傷。

筆記本的意義

遊戲世界的筆記本其實就是主角小時候用來記筆記的，在遊戲內主角似乎又回到了童年常常寫筆記的情況。筆記本封面底有張父親的照片，也代表著「筆記」是由父親教導給主角的習慣。

四、角色設定

（一）、主角

主角即為玩家操縱的角色，主角相關劇情可參考「二、（一）、劇情發展」的主角人生經歷。如同機制設定，主角也有兩面（正面／反面），玩家可以在遊戲中進行切換。

  • 正面主角：對應到現實世界中那個成年的主角人格。因為每天加班卻沒有目標，也找不到人生的意義。在日復一日的重複下逐漸麻痺。形象為三十初的上班族，面容憔悴且不甚開心，走起路來毫無生氣，感覺只是一個機器。穿著上班制服。第一關出場時只想趕快找到回家的路，因為明天還要上班。
  • 反面主角：對應到現實世界中主角的童年人格。童年因為有父親的加持，因此有屬於自己的夢想與興趣。形象在身高上與正面主角相同，但是面容上卻相當地有朝氣、紅潤。穿搭上則較為年輕。

（二）、工頭／廚師

工頭（正面）

形象為頭戴著工地用安全帽的中年大叔，手會輪流拿著各式工具。個性懶散、嗜睡、不負責任。基本上就是一個不愛工作、只想偷懶的傢伙，時常看到他在打瞌睡。說話說到一半可能直接睡著，話題容易圍繞在同一件工作上，並不斷地逃避。第二關初甚至走出工廠外偷懶，但卻因為側門被反鎖而無法回到工廠內。

象徵成年主角面對著大量工作時想逃避的心態，知道自己有很多工作要做，但還是選擇用睡眠來逃避。

廚師（反面）

形象為頭戴著廚師帽，手上有很多甜食，他也會時不時地舔著這些甜食。個性開朗、專心、很負責任。第二關為了準備宴會餐點，還自己去打獵，跟工頭形成對比。專心也和工頭形成對比，工頭連一件事都做不好。

象徵童年主角愛吃與能夠專注的性格，主角童年能夠專注在學習父親傳授的飛行知識，並追求夢想。

（三）、保全／騎士

保全（正面）

形象為穿著保全制服的白髮老人（未來老人）。個性悲觀、被動、不想服從老闆的命令。雖然職責是看管工廠的安全，但卻只是佔著位置、頹廢度日。是一個對自己的工作沒有熱情的傢伙，也會對工作內容有許多埋怨。

象徵成年主角對現有工作的不滿意、抱怨。抱怨對現有工作和上司的不滿意，時常說些悲觀的話。

騎士（反面）

形象為穿著盔甲的泰迪熊。個性忠誠（和保全的不忠形成對比）、剛毅木訥、果決。說話直接，而且也會時常表示自己必須擔保國王的安危。第二關當主角來到城堡時，也是最對身為外人的主角抱持懷疑的角色，因為身負守衛城堡的重任。

象徵童年主角兒時的玩伴，因為沒有其他朋友，只能跟玩具熊作伴，而玩具熊也忠實地守在主角身旁。

（四）、研究員／樂師

研究員（正面）

形象為半人半機器人，有很多機器手臂拿著零件或是研究用道具。個性保守、拘謹、遵守秩序，但是對於工廠的人偶很了解而且會告訴主角很多知識（提示）。會說一些專業的事情、就事論事，不會去提到其他瑣碎的事。第二關會給予所有正面道具的提示，也會談到關於拓印的事情。

象徵成年主角在現有體制下只能乖乖遵守，逼自己跟著這套體制去成為一個有秩序的人，但是沒什麼自主性。

樂師（反面）

形象為有很多手（六隻手臂）拿著不同的樂器（兩隻：小提琴、兩隻：管樂器、兩隻：鼓棒），有一隻猴子寵物（正面是小隻的機器人）負責敲鈸，身體設計成玩具的感覺。個性樂觀、崇尚自由、不受拘束（和研究員成為對比），樂於演奏出屬於自己的音樂。喜歡交談，但是一談到自己有興趣的事物甚至會不顧他人，進入自己的異想世界。可能會講一些其他事情，並提到關於這座城堡的運作，對於一般人覺得尋常的事也會感到很新奇。

象徵童年主角小時候的玩心，有很多樂器跟手臂表示甚麼都想玩，也有對新知好奇的心理。

（五）、廠長／國王

廠長（正面）
  • 形象為矮小的廠長（對應反面國王是小孩子）。個性上刁鑽、小心眼，期望部下都服從自己並好好工作，控制慾強（如同心魔佔據主角的心靈）。第二關不甚搭理主角，時常對自己的部下有不少抱怨。
  • 象徵主角踏入社會後接觸的工作體制，在體制下主角為了生存只能屈服，即便制度是不合理的。如同這座工廠以人偶生產人偶，毫無意義的生產是不合理的，但部下們也只能遷就。
國王（反面）
  • 形象為帶著王冠的小孩子，雖然是國王但卻不是穿國王華麗的服飾，而是一般的TShirt和短褲。容貌上是主角小時候的樣子。個性如同主角的童年，好奇、開朗、聰穎。第二關知道主角會來到這裡，因此想舉辦一個派對宴請主角，並考驗主角是否能夠會想起這個「自己」。第二關底會提示主角應該去查看「父親的畫」。
  • 象徵童年的主角，那個在精神世界內，被遺忘的玩心與夢想。

（六）、父親／林德柏格

父親（正面）
  • 主角「主角記憶中的父親」，畢竟現實的父親已往生。
  • 父親相關劇情可參考「二、（一）、前導劇情」，以下僅說明關於人物的設定：職業為飛機工程師，也是一名業餘的飛行員。擁有一架小型的私人民用飛機，四處飛行探索世界就是他的興趣之一。
  • 熟悉航空相關的各種科學知識，並時常教導童年主角知識，主角筆記的習慣也是源自於父親（寫飛行日誌）。
  • 會開飛機帶著小時候的主角與母親一同出遊，但最後因飛行意外不幸身亡。
  • 第三關主角遇到父親時，他並不認得主角。因為主角已經長大（容貌改變），所以他只覺得眼熟而已。以為主角也是飛行相關人士，因此使用提攜後輩說話的態度來告訴主角要做些什麼。
  • 形象為戴著飛行護目鏡與飛行服，容貌上和主角有些相似（畢竟是父親）。
  • 年齡約三十初（主角小時候）。個性成熟、慈祥、博學、負責任且愛護家庭，知道自己的夢想是什麼，失事時為了保護家人而犧牲自己（黃金精神）。
林德伯格（反面）
  • 原型為現實世界中的林德伯格（維基百科）。
  • 遊戲世界中，是個已退休的飛行員，平時依靠修理／打撈飛機來賺點小錢。由於調查到附近海域似乎有飛機墜毀過，因此前來打撈（其實就是主角小時候父親失事的飛機）。
  • 主角因為接觸過飛行知識而認識的人物，所以出現在主角的精神世界中。
  • 第三關遇見主角後，會提出請主角幫忙打撈飛機的請求。事後主角發現飛機是自家的，會懇求林德伯格把飛機讓給他。最後林德伯格幫主角把飛機修好。
  • 形象為穿著工作衣的退休飛行員（駕駛船來打撈飛機）。
