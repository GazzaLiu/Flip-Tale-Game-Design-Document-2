四、硬幣系統

當某些條件達成時，玩家可以獲得硬幣（白露里智奧的貨幣），玩家可以在最後一關使用硬幣購買商品，以決定最終結局的走向。獲得硬幣機制與顯示如下所示：

其中五個編號的硬幣獲得時機為：

  1. A：［第一關，非必得］當玩家手持花對關卡最左墓碑互動後（獻花）。
  2. B：［第一關，必得］搭乘熱氣球時（熱氣球組裝完畢，按互動搭乘前彈出）。
  3. C：［第二關，必得］在密室打倒心魔後，播完打敗動畫後彈出。
  4. D：［第三關，必得］在海底對飛機互動後（打撈飛機）彈出。
  5. E：［第三關，非必得］解完謎題B後（拼字謎題），書本掉下後彈出。
